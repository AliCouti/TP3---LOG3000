# Tests Unitaires - Flask Calculator

Ce dossier contient la suite de tests automatisés pour valider la logique mathématique de l'application à travers les opérateurs utilisés par le projet web Flask Calculator.

## Prérequis
- Python 3.x installé

## Exécution des tests
Pour lancer tous les tests, exécutez la commande suivante depuis la racine du projet :

**Sous Windows :**
```bash
python -m unittest discover tests
