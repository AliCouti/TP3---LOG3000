# Module : static

## Raison d’être

Ce module contient le fichier de style .CSS qui définit l'apparence de la page HTML
de l’application Flask.

Il aide à l'affichage de la calculatrice dans le navigateur et gère l'affichage dynamique lié aux interactions avec la souris (ex : survoler ou cliquer sur un bouton).

---

## Contenu principal

- `style.css`  
  Fichier de style principale de l’application.  
  - Permet l'affichage de la calculatrice.
  - Définit le thème et le positionnement des éléments HTML de la page principale.
