# Module : src

## Raison d’être

Ce module contient la logique principale de l’application Flask.
Il définit les routes, traite les requêtes et perfome les calculs nécessaire.

## Contenu principal

- `app.py`  
  Coeur de l'application Flask. Point d’entrée. Définit les routes HTTP et traite les requêtes de calcul.

- `calculator.py`  
  Contient la logique des opérations de calcul.

## Dépendances et hypothèses

- Dépend de Flask.
- Dépend du fichier `index.html` situés dans le dossier `/templates`.
